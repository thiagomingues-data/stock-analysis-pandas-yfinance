# 📈 Brazilian Stock Market Analysis (PETR4 vs ITUB4)

## 🇧🇷 Português

Este projeto realiza uma análise comparativa entre ações de empresas brasileiras utilizando dados reais da bolsa de valores.

As ações analisadas são:
- Petrobras (PETR4)
- Itaú (ITUB4)

---

### 📊 Objetivos

- Obter dados financeiros via API yfinance
- Analisar comportamento dos preços
- Comparar desempenho entre duas ações
- Visualizar variações e tendências

---

### 🛠️ Tecnologias utilizadas

- Python
- yfinance
- Pandas
- Matplotlib

---

### 📈 Visualizações

- Gráfico de linha (preço ao longo do tempo)
- Histograma de variação diária

---

### 🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar análise de dados financeiros, manipulação de dados com Pandas e criação de visualizações com Matplotlib.
